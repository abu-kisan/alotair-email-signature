# قالب توقيع البريد الإلكتروني — مجموعة العطير العقارية

القالب الرسمي المعتمد لتوقيع البريد، عربيّ باتجاه RTL، متوافق مع **Gmail** و **Outlook**.
كل شيء في ملف واحد: **`index.html`**.

---

## الملفات في هذا المجلد

| الملف | الغرض |
|---|---|
| **`index.html`** | القالب المعتمد — المعاينة + زر النسخ + توثيق الخط والألوان ومكان الشعار وأماكن المعلومات. **ابدأ من هنا.** |
| `logo-wordmark.png` | الشعار الأفقي (نصّي) — المستخدم في التوقيع. |
| `logo.png` | علامة الشعار الكحلية (الرمز فقط). |
| `logo-gold.png` | علامة الشعار الذهبية (للخلفيات الكحلية). |
| `ds-base.js` | يحمّل خطوط وألوان نظام التصميم لعرض الصفحة. |

---

## طريقة الاستخدام

### ١. عدّل المعلومات
افتح `index.html` في محرّر نصوص. كل قيمة قابلة للتعديل عليها تعليق بين قوسين، مثل `<!-- [الاسم] -->`.
استبدل النصوص وروابط `tel:` / `mailto:` / `href` بمعلومات الموظف. خريطة كاملة لمواضع الحقول موجودة داخل الصفحة في قسم **«مكان الشعار وأماكن المعلومات»**.

### ٢. ارفع الشعار (خطوة ضرورية)
البريد لا يعرض صورةً من جهازك المحلي. ارفع `logo-wordmark.png` على موقع المجموعة أو على Google Drive برابط عام، ثم ضع الرابط في خاصية `src` الخاصة بالشعار:
```html
<img src="https://www.alotair.com/email/logo-wordmark.png" ...>
```

### ٣. انسخ التوقيع
افتح `index.html` في المتصفح، واضغط **«نسخ التوقيع»** — أو حدّد كتلة التوقيع يدويًا وانسخها (`Ctrl/Cmd + C`).

### ٤. الصقه في بريدك
- **Gmail:** الإعدادات ⚙ ← عرض جميع الإعدادات ← عام ← التوقيع ← إنشاء جديد ← الصق ← **حفظ التغييرات**.
- **Outlook (ويب):** الإعدادات ⚙ ← Mail ← Compose and reply ← الصق في مربع التوقيع ← **Save**.
- **Outlook (كمبيوتر):** File ← Options ← Mail ← Signatures… ← New ← الصق ← **OK**.

## مواصفات الهوية المستخدمة

**الخط:** عربي = Helvetica Neue LT Arabic · لاتيني = Roboto · احتياطي = Tahoma.

**الألوان:**
| اللون | القيمة | الاستخدام |
|---|---|---|
| كحلي | `#0B1D3B` | العناوين والأسماء |
| ذهبي | `#AC975F` | المنصب والفواصل |
| فحمي | `#373737` | نص المعلومات |
| رمادي | `#6F7A88` | نص ثانوي |
| عاجي | `#F3F2ED` | الخلفيات |

**الشعار:** في العمود الأيمن، يفصله خط ذهبي عمودي عن المعلومات. حافظ على مساحة فراغ حوله.

## قواعد مهمة

- لا تُغيّر بنية الجدول أو التنسيق الداخلي (inline) — هكذا يبقى متوافقًا مع عملاء البريد.
- **أيقونات التواصل:** مرسومة كدوائر كحلية برموز بيضاء (SVG مضمّن). بعض عملاء البريد لا تعرض SVG؛ للموثوقية الكاملة استبدلها بصور PNG دائرية مستضافة برابط عام.
- لا تستخدم خطوطًا خارجية ولا JavaScript داخل كتلة التوقيع.
- عرض التوقيع 600px ويتقلّص تلقائيًا على الجوال.
- زر النسخ في الصفحة لا يدخل ضمن التوقيع المنسوخ — هو لراحة الاستخدام فقط.
